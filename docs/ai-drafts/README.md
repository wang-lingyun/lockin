# AI Drafts

Claude-generated planning and design drafts live here **before** human review.

Nothing in this folder is authoritative. Once a draft is reviewed and accepted, its
good parts are promoted into `../specs/` (for specs) or `../adr/` (for decisions), and
the draft may be left here for history.

Planned drafts (not yet written — foundation pass only):

- `architecture-draft.md` — Supabase schema, RLS strategy, API/server-action surface,
  page→component map.
- `stage-plan.md` — Stage 0–8 implementation plan with per-stage acceptance criteria.
- `ux-draft.md` — page layouts and component inventory.
